# create project!

1. Node.js설치

2. Vue.js 설치
```
npm install vue
```

3. Vue CLI 설치
```
npm install -g @vue/cli
```

4. Create Vue project
```
vue create {project name}
```

5. run server
```
npm run serve
```

6. exit
```
ctrl + c
```
