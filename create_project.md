# create project!

1. node.js설치

2. vue 설치
```
npm install -g @vue/cli
```

3. 확장프로그램 설치
```
vue 3 snippets
volar
HTML CSS Support
```

4. vue프로젝트 생성
```
vue create {project name}
```

5. run server
```
npm run serve
```

6. exit
```
ctrl + c
```
